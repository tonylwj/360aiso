# 360 智搜 Skill

基于360搜索引擎 API 的实时中文网页搜索工具，专为 LLM 和 AI Agent 设计。

## 为什么选择 360 智搜

- **全网内容 + 动态语义长摘要**——比传统短摘要语义更齐全，需要时可提供全文
- **时效性与权威性专项优化**——AI 语义相关度重排
- **精品知识库**——支持段落级语义查询与领域查询（如旅游、健康医疗等）
- **可信内容源查询**——支持指定可信内容源，配备多级内容安全库
- **指定站点查询**——具备定向抓取和快速补录能力
- **时间范围过滤**——支持指定最近 N 天或自定义时间范围
- **及时信息查询**——支持天气、汇率、股票等，内含新闻垂搜、百科、AI 总结等优质结果
- **多模态查询**——支持文搜图、以图搜图，可提供本地快照，防死链
- **接口内置大模型能力**——可做子 query 并行搜索、综合重排，乃至大模型总结
- **垂类搜索**——提供图搜、新闻、热点资讯等垂直搜索

## 运行要求

| 要求 | 说明 |
|------|------|
| API 密钥 | `SEARCH_360_API_KEY` |
| 系统工具 | `curl`（macOS 和主流 Linux 系统预装） |
| 网络 | 可访问 `api.360.cn` 的出站 HTTPS 连接 |

## 使用示例

安装并配置完成后，直接在对话框输入：

- "帮我搜索今天的 AI 行业新闻"
- "查一下中国新能源汽车市场最新动态"
- "联网搜索华为最近发布的新产品"
- "找一下最新的数据安全相关法规"
